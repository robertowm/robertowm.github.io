# Roberto Weidmann Menezes - Personal Page

Personal page and blog of Roberto Weidmann Menezes, built with [Eleventy](https://www.11ty.dev/) following a Brutalist design.

## 🚀 Quick start

1.  **Install dependencies.**

    ```sh
    npm install
    ```

1.  **Run in dev mode.**

    ```sh
    npm start
    ```

    It is now running at `http://localhost:8080`!

1.  **Build for production.**

    ```sh
    npm run build
    ```

    The generated site will be in the `dist/` directory.

## 🧐 What's inside?

A quick look at the top-level files and directories:

    .
    ├── src/            # Source files (Nunjucks, Markdown, etc.)
    │   ├── _data/      # Global data files (JSON/JS)
    │   ├── _includes/  # Layouts and partials
    │   └── posts/      # Blog posts
    ├── static/         # Static assets (images, icons, etc.)
    ├── .eleventy.js    # Eleventy configuration
    ├── .gitignore      # Files to ignore in Git
    ├── LICENSE         # MIT License
    ├── package.json    # Project manifest
    └── README.md       # This file

## 🚢 Deployment

The site is automatically deployed to GitHub Pages via GitHub Actions whenever changes are pushed or merged into the `dev` branch.
